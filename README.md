# Todo List

Простой **Todo List**, который позволяет отслеживать выполнение задач и время их выполнения.

## Функционал

- Создание, удаление задач
- Отслеживание времени выполнения задач
- Ставить на паузу выполнение задач
- Поиск задача по имени или описанию
- Статусы задач: `inProgress`, `done`, `start`, `pause`

## Технологии

- [Next.js](https://nextjs.org/)
- [Redux Toolkit](https://redux-toolkit.js.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Hook Form](https://react-hook-form.com/)
- FSD (Feature-Sliced Design) архитектура

## Установка и запуск

```bash
# Установить зависимости
npm install

# Собрать проект
npm run build

# Запустить сервер
npm run start
```
